# BetVision AI

## Scop
Platformă AI pentru analiză sportivă bazată pe date.

## Funcții MVP

- Analiză meciuri
- Confidence Score
- Risk Score
- Explicații AI
- Istoric rezultate
- Dashboard utilizator

## Flux AI

Date meci
→ Analiză
→ Predicție
→ Risc
→ Recomandare
→ Învățare

## Obiectiv

Construirea unei platforme AI de analiză sportivă.
