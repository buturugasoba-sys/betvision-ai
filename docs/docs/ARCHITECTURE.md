# BetVision AI Architecture

## Structura proiectului

frontend/
- Interfață utilizator
- Pagini web
- Dashboard

backend/
- API
- Autentificare
- Gestionare date

ai-engine/
- Analiză meciuri
- Confidence Score
- Risk Score
- Predicții

database/
- Utilizatori
- Echipe
- Meciuri
- Predicții
- Rezultate

## Flux aplicație

Utilizator
↓
Frontend
↓
Backend API
↓
AI Engine
↓
Database
↓
Rezultat analiză

## Obiectiv

O platformă AI care analizează meciuri și oferă explicații bazate pe date.
